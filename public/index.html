About the app:

This app was built to perform the loading of departments, jobs, and employees data, having as a data source CSV files.

Indications:

  • Use the 'Choose file' button to choose the CSV files
  • If you are going to load only employees data, make sure you have previously loaded departments and jobs data.

Additional notes:

  • CSV files do not have headers
  • There is 1 CSV file for each table
  • For loading the employees data, make sure that the ids of departments and jobs really exist (i.e., that those ids are in the departments and jobs data) and that they are not empty in the CSV file
